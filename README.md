# dotfiles
dotfiles

Install GNU `stow` and run
```
$ ./apply.sh
```
**OR**

Use `stow` to create symlinks to dotfiles with command:
```
$ stow <package name>
```
for each relevant package/directory in the root. Trailing slash omitted.

Currently:
```
$ stow emacs
$ stow jetbrains
$ stow ranger
$ stow tmux
$ stow vim
$ stow zsh
```
